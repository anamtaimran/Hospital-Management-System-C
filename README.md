# Hospital Management System 🏥

## About
A console-based Hospital Management System built in C
as my Semester 1 final project at IOBM University.

## Features
- Add new patient with ID, name, age and disease
- Display all patient records
- Search for a specific patient
- Update patient information
- Delete patient records
- Menu-driven interface

## Language
C Programming

## Concepts Used
- Arrays
- Functions
- Switch-case logic
- Loops
- String handling

## What I Learned
This was my first ever real project. It taught me
how to break a big problem into small functions,
how to use arrays to store data.
